采用红黑树的自然排序：
排序啊啊啊


概述:TreeSet集合底层才有红黑树算法,会对存储的元素默认使用自然
排序(从小到大),此时必须保证TreeSet集合中的元素对象是相同的数据
类型,负责报错

实现自然排序要求

  • TreeSet中的对象必须重写equals和HashCode方法,用于确保添加的对象不会重复
  • TreeSet集合元素实现java.util.Comparable接口
  • TreeSet调用集合元素的comparaTo方法来比较元素的大小关系,然后将集合元素按照升序排列

Comparable接口:(可比较的)

  1. 覆盖public int comparaTo(Object o)方法,在该方法中编写比较规则
  2. 在该方法中,比较当前对象(this)和参数对象o作比较(严格上说是比较对象中的数据,比如按照对象的年龄进行排序)
    this > o :返回正整数 1
    this < o :返回负整数 -1
    this == o :返回0 (此时认为两个对象是同一个对象)
